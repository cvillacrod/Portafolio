Web en WordPress + plugin personalizado para reservas online

Sitio corporativo en WordPress con plugin personalizado a medida para la gestión integral de reservas, planificación diaria y control de preautorizaciones de pago mediante WooCommerce.

El proyecto incluyó el desarrollo de una página web en WordPress con un sistema de reservas totalmente personalizado, que permite a los clientes reservar mesas online de forma sencilla y segura.

Se implementaron funcionalidades específicas adaptadas al flujo de trabajo del restaurante, como un planning visual de reservas, gestión de mesas y festivos, y la automatización de cobros con preautorizaciones. Todo ello optimizando tanto la operativa interna como la experiencia del usuario final.

Plugin personalizado de reservas

  • Listado de reservas con búsqueda y filtros por fecha/estado.
  • Crear, editar, confirmar y cancelar reservas desde el panel.
  • Estados con etiquetas visuales (pendiente, confirmada ycancelada).

  • Vista de reservas por franja horaria y mesa.
  • Colores por estado
  • Muestra la capacidad de la mesa ocupada.
  • Bloqueo por fechas u horarios especiales.

  • Listado de mesas con capacidad y estado (activa/bloqueada).
  • Calendario de festivos para deshabilitar reservas automáticamente.

  • La reserva crea un pedido en WooCommerce y realiza una preautorización de pago (p.ej. Redsys/TPV).
  • Desde el listado del plugin:
    • Si el cliente acude: se anula la preautorización (no se cobra).
    • Si el cliente no se presenta: se captura el cargo de la preautorización.
