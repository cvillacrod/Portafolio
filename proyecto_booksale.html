BOOKSALE

Aplicación móvil para intercambio y compraventa de libros

BOOKSALE es una aplicación Android desarrollada para facilitar el intercambio y la compraventa de libros entre estudiantes y exalumnos.

Hasta ahora este proceso se realizaba manualmente mediante carteles en tablones de anuncios. Con esta app, los usuarios pueden buscar, publicar y gestionar libros de manera sencilla desde su móvil.

El sistema integra filtros de búsqueda, chat entre usuarios, gestión de perfil y notificaciones, ofreciendo una experiencia más rápida, organizada y segura.

Estructura principal
  • Login / Registro
  • Menú principal
  • Buscar libro
  • Publicar libro
  • Mis libros
  • Historial / Chats
  • Menú usuario
  • Perfil
Funcionalidades principales

  • Filtros por título, ciclo, módulo, estado y precio.
  • Listado de libros disponibles ordenados por fecha de publicación.
  • Acceso directo al chat con el propietario del libro.

  • Formulario de publicación con ISBN, título, edición, ciclo y módulo.
  • Subida opcional de la portada del libro.
  • Gestión del estado: en venta, vendido, modificado o eliminado.

  • Listado de libros propios con estado visual (verde = en venta, rojo = vendidos).
  • Opciones para editar, vender o eliminar libros.
  • Chat con los interesados en cada publicación.

  • Historial de conversaciones filtrado por libros propios o de otros usuarios.
  • Notificaciones de mensajes no leídos.
  • Chat individual en tiempo real gracias a Firebase.

  • Registro y edición de perfil (nombre, apellidos, teléfono, contraseña).
  • Baja de usuario con eliminación de libros asociados.
  • Cierre de sesión seguro.
Stack técnico
Android Studio (Java) PHP MySQL Firebase (Mensajería) MVC
Resultado
  • Digitalización de un proceso que antes era manual en tablones de anuncios.
  • Mayor visibilidad y rapidez en la búsqueda de libros.
  • Conexión directa entre compradores y vendedores con chat en tiempo real.
  • Mejor organización y experiencia para los estudiantes.
